# GeNNre-Classifier
GeNNre Classifier implements a deep learning pipeline to solve the challenge where a single film belongs to multiple genres simultaneously. Moving beyond traditional baselines , this project utilizes a 2-layer Bidirectional LSTM with task-specific embeddings to capture the contextual narrative flow of plot summaries. For APS360 at UofT.
